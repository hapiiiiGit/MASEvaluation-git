This is the repository for MASs evaluation.

* analysis: using an llm-as-a-judge pipeline for evaluating coding phase and information transmission.
* Logs: the execution logs of three datasets
  * part1: HumanEval
  * part2: MBPP
  * part3: Project-level dataset
* MAS: the files for 6 MASs automatically analysis.
* Prompt：the prompts for llm-as-a-judge pipelines.
* RQ1 and RQ2: the additional code for evaluation.
* Task: the humanEval, MBPP and project-level datasets tasks.
* Run: the python scripts to automatically run MAS. Note: to get the resource consumption and ablation studies, we modify the MAS source code.
