def bubble_sort(arr):
    n = len(arr)
    # Step 2: Traverse through all array elements.
    for i in range(n):
        # Step 3: Last i elements are already in place.
        for j in range(0, n-i-1):

Shell Sort is a comparison-based sorting algorithm that generalizes insertion sort by allowing the exchange of items that are far apart. The idea is to arrange the list of elements so that, starting anywhere, taking every hth element produces a sorted list. Such a list is said to be h-sorted. The algorithm uses a sequence of gaps (called the "gap sequence") to sort elements that are far apart, and then reduces the gap until it becomes 1, at which point the list is sorted using a final insertion sort.

High-Level Steps:
1. Choose a gap sequence (commonly n//2, n//4, ..., 1).
2. For each gap, perform a gapped insertion sort:
   - Compare elements that are gap distance apart and swap if necessary.
   - Continue this process for all elements in the array.
3. Reduce the gap and repeat the process.
4. When the gap is 1, perform a final insertion sort pass.

Advantages:
- Shell Sort improves on insertion sort by moving elements over larger gaps, reducing the total number of movements required.
- It is efficient for medium-sized arrays and is easy to implement.

Use Cases:
- When you need a simple, in-place sorting algorithm that performs better than bubble, insertion, or selection sort for larger arrays.
- When memory usage is a concern, as Shell Sort is an in-place algorithm.

Complexity:
- The time complexity depends on the gap sequence, but it is generally better than O(n^2) for practical gap sequences.

def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr
