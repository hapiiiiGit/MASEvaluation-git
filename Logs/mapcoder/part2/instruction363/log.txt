Tutorial: Brute-force Algorithm for Uniformity Checking

Brute-force is a straightforward approach where you check every element in a collection to see if it meets a certain condition. For uniformity checking (e.g., all characters, elements, rows, or words are the same), the brute-force method involves:

1. Selecting a reference element (usually the first element).
2. Iterating through the entire collection.
3. Comparing each element to the reference.
4. If any element differs, you immediately conclude the collection is not uniform.
5. If the iteration completes without finding a difference, the collection is uniform.

This approach is simple and effective for small to moderate-sized collections. It does not require any advanced data structures or algorithms, and is easy to implement and understand. The time complexity is O(n), where n is the number of elements in the collection.

Use brute-force when:
- The collection size is not extremely large.
