Frequency Counting with Hash Maps (Dictionaries)
-------------------------------------------------
This algorithm is useful for problems where you need to know how many times each element appears in a collection (list, array, etc.). The general steps are:

1. **Initialize a hash map (dictionary):** This will store each unique element as a key and its count as the value.
2. **Traverse the collection:** For each element, increment its count in the hash map.
3. **Post-processing:** After counting, you can easily query the hash map for elements that meet certain criteria (e.g., appear only once, appear more than once, etc.).
4. **Apply the criteria:** For the original problem, you would sum all elements whose count is exactly one.

This approach is efficient because:
- Counting is done in O(n) time, where n is the number of elements.
- Hash maps provide constant time access and update for each element.

def sum_of_non_repeated_elements(lst):
    freq = {}
    for num in lst:
        freq[num] = freq.get(num, 0) + 1
    return sum(num for num, count in freq.items() if count == 1)
