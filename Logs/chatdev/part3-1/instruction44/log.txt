from flask import Flask, jsonify
from shopify_api import fetch_daily_orders
import datetime

def create_app():

    @app.route('/order_count', methods=['GET'])

```python
'''
Module to handle interactions with the Shopify Admin API.
'''

import requests
from config import SHOPIFY_API_KEY, SHOPIFY_PASSWORD, SHOPIFY_STORE_NAME
import datetime

def authenticate():
    '''
    Authenticates with the Shopify API using provided credentials.
    '''
    return (SHOPIFY_API_KEY, SHOPIFY_PASSWORD)

def fetch_daily_orders():
    '''
    Fetches the daily orders based on the store's timezone.
    '''
    auth = authenticate()
    today = datetime.datetime.now().strftime('%Y-%m-%d')
    url = f"https://{SHOPIFY_STORE_NAME}.myshopify.com/admin/api/2023-10/orders.json?created_at_min={today}T00:00:00Z&created_at_max={today}T23:59:59Z"
    
    response = requests.get(url, auth=auth)
    if response.status_code == 200:
        orders = response.json().get('orders', [])
        return len(orders)
    else:
        return 0
```

'''
Main application to create an HTTP endpoint for order count.
'''
from flask import Flask, jsonify
from shopify_api import fetch_daily_orders
import threading
import time

app = Flask(__name__)

@app.route('/order_count', methods=['GET'])
def order_count():
    '''
    Endpoint to get the daily order count.
    '''
    count = fetch_daily_orders()
    return jsonify({'order_count': count})

def reset_order_count():
    '''
    Resets the order count at midnight.
    '''
    while True:
        now = datetime.datetime.now()
        midnight = (now + datetime.timedelta(days=1)).replace(hour=0, minute=0, second=0, microsecond=0)
        time_to_midnight = (midnight - now).total_seconds()
        time.sleep(time_to_midnight)

if __name__ == '__main__':
    threading.Thread(target=reset_order_count, daemon=True).start()
    app.run(host='0.0.0.0', port=5000)

This application provides an HTTP endpoint that authenticates with the Shopify Admin API to count the daily orders in a store's timezone. It responds to requests with the order count in JSON format and resets the count at midnight.

## Main Functions

- **HTTP Endpoint**: The application exposes an endpoint `/order_count` that returns the number of orders placed on the current day in JSON format.
  
- **Authentication**: The application securely authenticates with the Shopify Admin API using provided API credentials.

- **Daily Order Counting**: The application fetches the count of orders created on the current day based on the store's timezone.

- **Midnight Reset**: The order count resets automatically at midnight to ensure accurate daily tracking.

## Installation Instructions

To set up the environment and install the required dependencies, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://your-repository-url.git
   cd your-repository-directory
   ```

2. **Create a Virtual Environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**:
   Ensure you have `pip` installed, then run:
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure API Credentials**:
   Open the `config.py` file and replace the placeholders with your actual Shopify API credentials:
   ```python
   SHOPIFY_API_KEY = 'your_api_key_here'
   SHOPIFY_PASSWORD = 'your_api_password_here'
   SHOPIFY_STORE_NAME = 'your_store_name_here'
   ```

## How to Use

1. **Run the Application**:
   Start the application by running the following command:
   ```bash
   python main.py
   ```
   The application will start and listen for incoming requests on port 5000.

2. **Access the Order Count**:
   To get the daily order count, send a GET request to the following URL:
   ```
   http://localhost:5000/order_count
   ```
   You can use tools like `curl`, Postman, or simply a web browser to access this endpoint.

3. **Response Format**:
   The response will be in JSON format, similar to the following:
   ```json
   {
       "order_count": 15
   }
   ```

4. **Resetting the Count**:
   The order count will automatically reset at midnight based on the server's timezone.

## Additional Notes

- Ensure that your server's timezone is correctly set to match your Shopify store's timezone for accurate order counting.
- This application is designed to be lightweight and can be hosted on any service that supports Python applications, such as Heroku, AWS Lambda, or DigitalOcean.
