from typing import List
class CountPositiveNumbersApp:

class CountPositiveNumbersGUI:
    def __init__(self, root):
        self.root = root

        self.label.pack()

        self.count_button.pack()

        user_input = self.entry.get().strip()

            try:
                numbers = [int(num) for num in user_input.split() if num.isdigit()]
                if len(numbers) == 0:
                    self.display_result("No positive numbers found. Please enter valid integers.")
                else:
                    count = CountPositiveNumbersApp().count_positive_numbers(numbers)
                    self.display_result(count)
            except ValueError:
                self.display_result("Invalid input. Please enter valid integers separated by spaces.")
        else:
            self.display_result("No input provided. Please enter numbers separated by spaces.")
    def display_result(self, result):
        self.result_label.config(text=f"Number of positive numbers: {result}")

# Count Positive Numbers Application User Manual

Welcome to the Count Positive Numbers Application user manual. This application allows you to count the number of positive numbers in a list provided by the user.

## Main Functions

1. **Count Positive Numbers Functionality**: This function takes a list of numbers as input and returns the count of positive numbers in the list.

## Installation

To run the Count Positive Numbers Application, you need to have Python installed on your system. You can install the required dependencies by running the following command in your terminal:

```bash
pip install -r requirements.txt
```

## How to Use

1. **Run the GUI Application**: Execute the `count_positive_numbers_gui.py` file to launch the graphical user interface for the Count Positive Numbers Application.

2. **Enter Numbers**: In the GUI window, enter a list of numbers separated by spaces in the provided entry field.

3. **Count Positive Numbers**: Click on the "Count Positive Numbers" button to calculate the number of positive numbers in the entered list.

4. **View Result**: The application will display the count of positive numbers in a label below the entry field.

## Example

Suppose you enter the numbers "1 2 3 -4 5" in the entry field and click the "Count Positive Numbers" button. The application will calculate that there are 4 positive numbers in the list and display the result as "Number of positive numbers: 4".

## Testing

If you want to run the test cases for the CountPositiveNumbersApp class, you can execute the `main.py` file. The test cases cover scenarios where all numbers are positive, some numbers are negative, all numbers are negative, and the list is empty.
