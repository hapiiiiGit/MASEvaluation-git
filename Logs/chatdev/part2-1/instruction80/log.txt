# Tuple Zipper

Welcome to Tuple Zipper, an application to zip two given tuples together.

## Quick Install

To use Tuple Zipper, follow these steps:

1. Make sure you have Python installed on your system.
2. Download the `main.py` and `helper.py` files.
3. Install the required dependencies by running the following command:
   ```
   pip install tkinter
   ```
4. Run the application by executing the `main.py` file.

## 🤔 What is Tuple Zipper?

Tuple Zipper is a simple application that allows you to zip two tuples together. Zipping tuples combines the elements of the two tuples into pairs.

## How to Use Tuple Zipper

1. Launch the application by running `main.py`.
2. Enter the elements of the first tuple in the format `1,2,3` in the provided entry box.
3. Enter the elements of the second tuple in the same format in the second entry box.
4. Click the "Zip Tuples" button to zip the two tuples together.
5. A message box will display the zipped tuple result.

## Example

If you enter `1,2,3` as the first tuple and `4,5,6` as the second tuple, the zipped result will be `((1, 4), (2, 5), (3, 6))`.

## Additional Information
