'''
Main application file for finding sequences of lowercase letters joined with an underscore.
'''

import tkinter as tk
from tkinter import messagebox
from utils import find_lowercase_sequences

class App:
    def __init__(self, master):
        self.master = master
        master.title("Lowercase Sequence Finder")

        self.label = tk.Label(master, text="Enter a string:")
        self.label.pack()

        self.entry = tk.Entry(master)
        self.entry.pack()

        self.find_button = tk.Button(master, text="Find Sequences", command=self.find_sequences)
        self.find_button.pack()

        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def find_sequences(self):
        user_input = self.entry.get()
        sequences = find_lowercase_sequences(user_input)
        if sequences:
            self.result_label.config(text=f"Found sequences: {', '.join(sequences)}")
        else:
            self.result_label.config(text="No sequences found.")

if __name__ == "__main__":
    root = tk.Tk()
    app = App(root)
    root.mainloop()
```

#### File: utils.py

```python
'''
Utility functions for processing strings to find sequences of lowercase letters joined with an underscore.
'''

import re

def find_lowercase_sequences(input_string):
    """
    Finds all sequences of lowercase letters joined with an underscore in the given input string.
    
    Args:
        input_string (str): The string to search for sequences.
        
    Returns:
        list: A list of found sequences.
    """
    pattern = r'\b[a-z]+(?:_[a-z]+)*\b'
    return re.findall(pattern, input_string)
```

'''

import re
def find_lowercase_sequences(input_string):
    """
    Finds all sequences of lowercase letters joined with an underscore in the given input string.
    Args:
        input_string (str): The string to search for sequences.
    Returns:
        list: A list of found sequences.
    """
    pattern = r'\b[a-z]+(?:_[a-z]+)*\b'
    return re.findall(pattern, input_string)

# Lowercase Sequence Finder

A simple application to find sequences of lowercase letters joined with an underscore in a given string.

## Quick Install

To run the application, you need to have Python installed on your machine. You can install the required dependencies using pip:

```bash
pip install tkinter
```

## 🤔 What is this?

The Lowercase Sequence Finder is a desktop application that allows users to input a string and find sequences of lowercase letters that are joined by underscores. This can be useful for validating or extracting specific patterns from text data.

## Main Functions

- **Input Field**: Enter any string in the provided input field.
- **Find Sequences Button**: Click this button to process the input string and find sequences of lowercase letters joined by underscores.
- **Result Display**: The application will display the found sequences or indicate if no sequences were found.

## How to Use

1. **Run the Application**:
   - Execute the `main.py` file in your terminal or command prompt:
   ```bash
   python main.py
   ```

2. **Enter a String**:
   - In the application window, you will see a label prompting you to enter a string. Type your desired string into the input field.

3. **Find Sequences**:
   - Click the "Find Sequences" button. The application will process your input and search for sequences of lowercase letters joined by underscores.

4. **View Results**:
   - The results will be displayed below the button. If sequences are found, they will be listed; if none are found, a message will indicate that.

## Example Usage

- Input: `this_is_a_test_string`
- Output: `Found sequences: this_is_a_test_string`

- Input: `No_Sequences_Here`
