```python
def difference_even_odd_digits(number):
    # Convert the number to a string to iterate through each digit
    digits = str(number)
    
    # Initialize sums for even and odd digits
    sum_even = 0
    sum_odd = 0
    
    # Iterate through each digit
    for digit in digits:
        # Convert the digit back to an integer
        digit = int(digit)
        
        # Check if the digit is even or odd and update the sums accordingly
        if digit % 2 == 0:
            sum_even += digit
        else:
            sum_odd += digit
            
    # Calculate the difference
    difference = sum_even - sum_odd
    return difference

# Example usage
result = difference_even_odd_digits(123456)
print("The difference between the sum of even and odd digits is:", result)
```

# Even Odd Digit Difference Calculator

Welcome to the Even Odd Digit Difference Calculator! This software allows you to find the difference between the sum of even and odd digits in a given number.

## Main Functions

The main function of this software is to calculate the difference between the sum of even and odd digits in a number provided by the user.

## Installation

To use the Even Odd Digit Difference Calculator, follow these steps:

1. Make sure you have Python installed on your system.

   ```

   python gui.py
   ```

6. Click the "Calculate" button to see the difference between the sum of even and odd digits displayed on the screen.
