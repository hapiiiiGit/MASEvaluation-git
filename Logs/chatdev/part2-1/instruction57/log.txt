To use the Opposite Sign Checker application, follow these steps to set up the environment:

2. Clone or download the project files from the repository.

3. Install the required dependencies by running the following command in your terminal:

```bash
pip install tkinter
```

## How to Use

Once you have installed the dependencies, you can run the Opposite Sign Checker application by following these steps:

1. Open a terminal window and navigate to the directory where you saved the project files.

2. Run the `main.py` file using the following command:

```bash
python main.py
```

3. The GUI window for the Opposite Sign Checker application will open.

4. Enter the first integer in the provided input field labeled "Enter first integer".

5. Enter the second integer in the input field labeled "Enter second integer".

6. Click on the "Check Signs" button to see if the two integers have opposite signs.
