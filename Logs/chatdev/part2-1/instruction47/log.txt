'''

from tkinter import Tk, Label, Entry, Button, StringVar
from bit_operations import set_odd_bits
class OddBitSetterApp:
    '''
    GUI application to set odd bits of a number.
    '''
    def __init__(self, master):
        self.master = master
        master.title("Odd Bit Setter")
        self.label = Label(master, text="Enter a number:")
        self.label.pack()
        self.number_var = StringVar()
        self.entry = Entry(master, textvariable=self.number_var)
        self.entry.pack()
        self.set_button = Button(master, text="Set Odd Bits", command=self.set_odd_bits)
        self.set_button.pack()
        self.result_label = Label(master, text="")
        self.result_label.pack()
    def set_odd_bits(self):
        '''
        Retrieves the number from the entry, validates input, sets odd bits, and displays the result.
        '''
        input_number = self.number_var.get()
        if input_number.isdigit():
            number = int(input_number)
            if number >= 0:
                result = set_odd_bits(number)
                self.result_label.config(text=f"Result: {result}")
            else:
                self.result_label.config(text="Please enter a non-negative integer.")
        else:
            self.result_label.config(text="Please enter a valid integer.")
if __name__ == "__main__":
    root = Tk()
    app = OddBitSetterApp(root)
    root.mainloop()

To run the Odd Bit Setter application, you need to have Python installed on your machine. You can download Python from [python.org](https://www.python.org/downloads/).

Once Python is installed, you can install the required dependencies using pip:

```bash
pip install tk
```

## 🤔 What is this?

The Odd Bit Setter application allows users to input a non-negative integer and sets all odd bits of that integer. This is useful for understanding bit manipulation and how binary representations work in programming.

### Main Functions

- **Input a Number**: Users can enter a non-negative integer into the application.
- **Set Odd Bits**: When the user clicks the "Set Odd Bits" button, the application processes the input number and sets all odd bits.
- **Display Result**: The application displays the result after processing the input.

## 📖 How to Use

1. **Launch the Application**:
   - Run the `main.py` file to start the application. You can do this from your terminal or command prompt:
   ```bash
   python main.py
   ```

2. **Enter a Number**:
   - In the GUI window, you will see a text box labeled "Enter a number:". Type a non-negative integer into this box.

3. **Set Odd Bits**:
   - Click the "Set Odd Bits" button. The application will validate your input and process the number.

4. **View the Result**:
   - The result will be displayed below the button, showing the integer with all odd bits set.

### Example

- If you enter `5` (which is `0b0101` in binary), the application will set the odd bits, resulting in `0b0101 | 0b010101010101...`, which will yield `0b0101` (5) with odd bits set, resulting in `0b1111` (15).
