'''
This file contains helper functions for substring calculation.
'''
def calculate_digit_sum(string):
    return sum(int(char) for char in string if char.isdigit())
def is_digit_sum_equal_length(string):
    # Check if the substring contains only digits
    if not string.isdigit():
        return False
    return calculate_digit_sum(string) == len(string)
def find_all_substrings(string):
    substrings = []
    for i in range(len(string)):
        for j in range(i+1, len(string)+1):
            substrings.append(string[i:j])
    return substrings
```

This file contains the main function to count the number of substrings with the sum of digits equal to their length.
'''
from helper import calculate_digit_sum, is_digit_sum_equal_length, find_all_substrings
def count_substrings_with_equal_sum_length(input_string):
    substrings = find_all_substrings(input_string)
    count = 0
    for substring in substrings:
        if is_digit_sum_equal_length(substring):
            count += 1
    return count

# Substring Counter

## Introduction

Welcome to the Substring Counter application! This software is designed to count the number of substrings within a given string where the sum of the digits in the substring is equal to the length of the substring.

## Main Functions

The main function of this software is to provide a graphical user interface (GUI) where users can input a string, and the application will calculate and display the number of substrings that meet the specified criteria.

## Installation

To use the Substring Counter application, you need to have Python installed on your system. You can install the required dependencies by running the following command in your terminal:

```bash
pip install -r requirements.txt
```

## How to Use

1. Clone the repository containing the Substring Counter application code.
2. Navigate to the directory where the code is located.
3. Run the `main.py` file using Python to launch the GUI.
4. Enter a string in the input field provided.
5. Click the "Submit" button to calculate the number of substrings with the sum of digits equal to their length.
6. The result will be displayed on the screen.

## Example

Suppose you input the string "abc123def456" into the application. The application will identify substrings such as "123" and "456" where the sum of the digits (1+2+3=6, 4+5+6=15) is equal to the length of the substring (3, 3), and count them accordingly.

 # Substring Counter

## Introduction

Welcome to the Substring Counter application! This software is designed to count the number of substrings within a given string where the sum of the digits in the substring is equal to the length of the substring.

## Main Functions

The main function of this software is to provide a graphical user interface (GUI) where users can input a string, and the application will calculate and display the number of substrings that meet the specified criteria.

## Installation

To use the Substring Counter application, you need to have Python installed on your system. You can install the required dependencies by running the following command in your terminal:

```bash
pip install -r requirements.txt
```

## How to Use

1. Clone the repository containing the Substring Counter application code.
2. Navigate to the directory where the code is located.
3. Run the `main.py` file using Python to launch the GUI.
4. Enter a string in the input field provided.
5. Click the "Submit" button to calculate the number of substrings with the sum of digits equal to their length.
6. The result will be displayed on the screen.

## Example

Suppose you input the string "abc123def456" into the application. The application will identify substrings such as "123" and "456" where the sum of the digits (1+2+3=6, 4+5+6=15) is equal to the length of the substring (3, 3), and count them accordingly.
