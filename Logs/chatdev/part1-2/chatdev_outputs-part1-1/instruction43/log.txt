'''

Function to calculate the sum of squares and cubes based on index rules.

'''

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0:
            total += value ** 3
        else:
            total += value
    return total
```

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not

    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0:
            total += value ** 3
        else:
            total += value
    return total

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 3 != 0:
            total += value ** 3
        else:
            total += value
    return total

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 3 != 0:
            total += value ** 3
        else:
            total += value
    return total

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 12 != 0:
            total += value ** 3
        else:
            total += value
    return total

def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 12 != 0:
            total += value ** 3
        else:
            total += value
    return total

'''
Function to calculate the sum of squares and cubes based on index rules.
'''
def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 3 != 0:  # Corrected condition
            total += value ** 3
        else:
            total += value
    return total

'''
Function to calculate the sum of squares and cubes based on index rules.
'''
def sum_squares(lst):
    """
    This function will take a list of integers. For all entries in the list, the function shall square the integer entry if its index is a 
    multiple of 3 and will cube the integer entry if its index is a multiple of 4 and not a multiple of 3. The function will not 
    change the entries in the list whose indexes are not a multiple of 3 or 4. The function shall then return the sum of all entries.
    Examples:
    For lst = [1,2,3] the output should be 6
    For lst = []  the output should be 0
    For lst = [-1,-5,2,-1,-5] the output should be -126
    """
    total = 0
    for index, value in enumerate(lst):
        if index % 3 == 0:
            total += value ** 2
        elif index % 4 == 0 and index % 3 != 0:  # Corrected condition
            total += value ** 3
        else:
            total += value
    return total

## Main Functions

The Sum Squares Application provides a simple graphical user interface (GUI) for users to input a list of integers and calculate the sum based on the following rules:
- Square the integer if its index is a multiple of 3.
- Cube the integer if its index is a multiple of 4 and not a multiple of 3.
- Leave the integer unchanged if its index is not a multiple of 3 or 4.

The application then returns the sum of all entries after applying these rules.
