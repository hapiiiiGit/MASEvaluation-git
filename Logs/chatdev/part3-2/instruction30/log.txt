def create_app():
    app = Flask(__name__)
    CORS(app)  # Enable CORS for all routes

from flask import Flask, request, jsonify
from flask_cors import CORS
from openai_client import OpenAIClient
def create_app():
    app = Flask(__name__)
    CORS(app)  # Enable CORS for all routes
    openai_client = OpenAIClient()
    @app.route('/chat', methods=['POST'])
    def chat():
        data = request.json
        prompt = data.get('prompt', '')
        response = openai_client.get_response(prompt)
        return jsonify({'response': response})
    return app

'''
Module to handle interactions with the OpenAI API.
'''
import openai
import os
from dotenv import load_dotenv
class OpenAIClient:
    def __init__(self):
        load_dotenv()  # Load environment variables from .env file
        openai.api_key = os.environ.get('OPENAI_API_KEY')
        if not openai.api_key:
            raise ValueError("OpenAI API key not found. Please set it in the .env file.")
    def get_response(self, prompt):
        try:
            response = openai.Completion.create(
                engine="text-davinci-003",
                prompt=prompt,
                max_tokens=150
            )
            return response.choices[0].text.strip()
        except Exception as e:
            return f"Error: {str(e)}"

```python
'''
Module to handle interactions with the OpenAI API.
'''
import openai
import os
from dotenv import load_dotenv
class OpenAIClient:
    def __init__(self):
        load_dotenv()  # Load environment variables from .env file
        openai.api_key = os.environ.get('OPENAI_API_KEY')
        if not openai.api_key:
            raise ValueError("OpenAI API key not found. Please set it in the .env file.")
    def get_response(self, prompt):
        try:
            response = openai.Completion.create(
                engine="text-davinci-003",
                prompt=prompt,
                max_tokens=150
            )
            return response.choices[0].text.strip()
        except Exception as e:
            return f"Error: {str(e)}"

from flask import Flask, request, jsonify
from flask_cors import CORS
from openai_client import OpenAIClient
def create_app():
    app = Flask(__name__)
    CORS(app)  # Enable CORS for all routes
    openai_client = OpenAIClient()
    @app.route('/chat', methods=['POST'])
    def chat():
        data = request.json
        prompt = data.get('prompt', '')
        response = openai_client.get_response(prompt)
        return jsonify({'response': response})
    return app
if __name__ == '__main__':
    app = create_app()
    app.run(host='0.0.0.0', port=5000)

'''
Module to handle interactions with the OpenAI API.
'''
import openai
import os
from dotenv import load_dotenv
class OpenAIClient:
    def __init__(self):
        load_dotenv()  # Load environment variables from .env file
        openai.api_key = os.environ.get('OPENAI_API_KEY')
        if not openai.api_key:
            raise ValueError("OpenAI API key not found. Please set it in the .env file.")
    def get_response(self, prompt):
        try:
            response = openai.Completion.create(
                engine="text-davinci-003",
                prompt=prompt,
                max_tokens=150
            )
            return response.choices[0].text.strip()
        except Exception as e:
            return f"Error: {str(e)}"

    return app

'''
Module to handle interactions with the OpenAI API.
'''
import openai
import os
from dotenv import load_dotenv
class OpenAIClient:
    def __init__(self):
        load_dotenv()  # Load environment variables from .env file
        openai.api_key = os.environ.get('OPENAI_API_KEY')
        if not openai.api_key:
            raise ValueError("OpenAI API key not found. Please set it in the .env file.")
    def get_response(self, prompt):
        try:
            response = openai.Completion.create(
                engine="text-davinci-003",
                prompt=prompt,
                max_tokens=150
            )
            return response.choices[0].text.strip()
        except Exception as e:
            return f"Error: {str(e)}"

# Chatbot Backend and Frontend Deployment Manual

Welcome to the user manual for deploying a Python-based chatbot backend on an Azure VM and interfacing it with a web frontend. This guide will walk you through the main functions of the software, installation of environment dependencies, and how to use the application.

## Overview

This application serves as a backend for a web-based chatbot that interfaces with the OpenAI API. It is designed to be deployed on an Azure Virtual Machine (VM) and accessed via a web app with a clean UI/UX. The backend is built using Flask, while the frontend is developed using HTML and JavaScript.

## Main Functions

- **Chatbot Backend**: A Flask application that handles HTTP requests and communicates with the OpenAI API to generate responses based on user input.
- **OpenAI API Integration**: Utilizes the OpenAI API to process prompts and return AI-generated text.
- **Web Frontend**: A simple and intuitive interface for users to interact with the chatbot, built using HTML and JavaScript.

## Installation

### Prerequisites

- An Azure account with access to create a Virtual Machine.
- Python 3.x installed on your local machine.
- Basic knowledge of deploying applications on Azure.

### Step-by-Step Installation

1. **Set Up Azure VM**:
   - Log in to your Azure account.
   - Create a new Virtual Machine with a basic configuration to minimize costs.
   - Ensure the VM has a public IP address for external access.

2. **Install Python and Git on Azure VM**:
   - SSH into your Azure VM.
   - Update package lists: `sudo apt update`.
   - Install Python: `sudo apt install python3`.
   - Install pip: `sudo apt install python3-pip`.
   - Install Git: `sudo apt install git`.

3. **Clone the Repository**:
   - Clone the project repository to your VM: `git clone <repository-url>`.

4. **Install Python Dependencies**:
   - Navigate to the project directory: `cd <project-directory>`.
   - Install dependencies: `pip install -r requirements.txt`.

5. **Configure Environment Variables**:
   - Create a `.env` file in the project directory.
   - Add your OpenAI API key: `OPENAI_API_KEY=your_actual_openai_api_key`.

6. **Run the Flask Application**:
   - Start the Flask server: `python3 main.py`.
   - Ensure the server is running and accessible via the VM's public IP.

## Usage

### Accessing the Web App

1. **Open the Web App**:
   - Open a web browser and navigate to `http://<YOUR_AZURE_VM_IP>:5000`.
   - Replace `<YOUR_AZURE_VM_IP>` with your actual Azure VM IP address.

2. **Interacting with the Chatbot**:
   - Type your message in the provided text area.
   - Click the "Send" button to submit your message.
   - The chatbot's response will be displayed below the input area.

## Troubleshooting

- **OpenAI API Key Error**: Ensure your `.env` file contains the correct API key.
- **Network Issues**: Verify that your Azure VM's network security group allows inbound traffic on port 5000.
- **CORS Issues**: Ensure CORS is enabled in the Flask application to allow requests from the web frontend.

## Conclusion

This manual provides a comprehensive guide to deploying and using the chatbot application. By following these steps, you can successfully set up a Python-based backend on Azure and interact with it through a web interface. For further assistance, please refer to the documentation or contact support.
