- Internet connection for downloading dependencies

### Quick Install

1. **Clone the Repository**

   Clone the repository to your local machine using the following command:

   ```bash
   git clone https://github.com/your-repo/support-warranty-system.git
   cd support-warranty-system
   ```

2. **Install Dependencies**

   Install the required Python packages using pip:

   ```bash
   pip install -r requirements.txt
   ```

   Alternatively, you can use conda:

   ```bash
   conda create --name support-warranty-system python=3.8
   conda activate support-warranty-system
   pip install -r requirements.txt
   ```

## How to Use

1. **Run the Application**

   Navigate to the directory where the code is located and run the main module:

   ```bash
   python main.py
   ```

2. **Using the GUI**

   - Once the application is running, a GUI window will appear.
   - The main screen will welcome you to the Support and Warranty System.
   - Follow the on-screen instructions to interact with the system.

3. **CRM and Helpdesk Operations**

   - The system will automatically connect to the CRM and fetch customer data.
   - Tickets will be created and responded to automatically based on the customer data.

4. **Communication**

   - Notifications will be sent to customers via email regarding ticket status.

5. **Warranty Evaluation**

   - The system will evaluate warranty claims and generate decisions.

6. **Data Security**

   - Customer data will be encrypted to ensure security.
   - Compliance checks will be performed automatically.

## Troubleshooting

- If you encounter any errors, the GUI will display an error message with details.
- Ensure all dependencies are installed correctly and the CRM/helpdesk API endpoints are accessible.

## Documentation

For more detailed documentation, please refer to the source code comments and function docstrings within each module.
