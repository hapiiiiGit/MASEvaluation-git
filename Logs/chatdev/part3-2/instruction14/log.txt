## 🤔 What is this?

Large language models (LLMs) are emerging as a transformative technology, enabling developers to build applications that they previously could not. However, using these LLMs in isolation is often insufficient for creating a truly powerful app - the real power comes when you can combine them with other sources of computation or knowledge.

This library aims to assist in the development of those types of applications. Common examples of these applications include:

**❓ Question Answering over specific documents**

- Documentation

- End-to-end Example: Question Answering over Notion Database

**🤖 Agents**

- Documentation

- End-to-end Example: GPT+WolframAlpha

# Quiz Automation Tool for macOS

Welcome to the Quiz Automation Tool for macOS! This tool is designed to automate the process of capturing and recognizing text from your screen using Optical Character Recognition (OCR) technology. It is equipped with a hotkey trigger and supports Retina scaling for high-resolution displays.

## Main Functions

1. **OCR Functionality**: The tool uses `pytesseract` to recognize text from images captured from your screen. It supports Retina scaling to ensure high-quality text recognition on macOS devices.

2. **Hotkey Trigger**: The tool allows you to set up a global hotkey (`Ctrl + Shift + O`) to trigger the OCR functionality instantly.

3. **Retina Scaling Support**: The tool adjusts for Retina scaling, ensuring that the captured images are processed at the correct resolution for accurate text recognition.

## Installation Guide

### Prerequisites

- **Python 3.x**: Ensure that Python is installed on your system. You can download it from [python.org](https://www.python.org/).
- **pip**: Python package installer, usually included with Python installations.

### Installation Steps

1. **Clone the Repository**: Clone the repository or download the source code to your local machine.

   ```bash
   git clone <repository-url>
   ```

2. **Navigate to the Project Directory**: Open a terminal and navigate to the directory where the source code is located.

   ```bash
   cd path/to/project-directory
   ```

3. **Install Required Packages**: Use pip to install the necessary Python packages listed in `requirements.txt`.

   ```bash
   pip install -r requirements.txt
   ```

4. **Install Tesseract OCR**: Since `pytesseract` is a wrapper for the Tesseract OCR engine, ensure Tesseract is installed on your macOS system. You can install it using Homebrew:

   ```bash
   brew install tesseract
   ```

## Usage Guide

1. **Running the Tool**: Execute the main script to start the quiz automation tool.

   ```bash
   python main.py
   ```

2. **Using the Hotkey**: Once the tool is running, use the global hotkey `Ctrl + Shift + O` to trigger the OCR functionality. The tool will capture the screen, process the image, and output the recognized text.

3. **Adjusting for Retina Displays**: The tool automatically adjusts for Retina scaling, so no additional configuration is needed for high-resolution displays.

## Additional Information

- **Advanced Image Processing**: If you require more advanced image processing capabilities, consider installing `opencv-python-headless`.

- **macOS Specific Dependencies**: The `pyobjc-framework-Quartz` package is included to assist with screen capture and handling Retina displays on macOS.

For further assistance or to report issues, please contact our support team or visit our GitHub repository for more information.
