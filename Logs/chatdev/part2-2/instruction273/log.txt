def is_min_heap(arr, n, i):
    # If a leaf node
    if i >= (n - 2) // 2:
        return True
    # Check if the current node is smaller than its left child
    if arr[i] > arr[2 * i + 1]:
        return False
    # Check if the current node is smaller than its right child, if it exists
    if 2 * i + 2 < n and arr[i] > arr[2 * i + 2]:
        return False
    # Recursively check the left and right subtrees
    return is_min_heap(arr, n, 2 * i + 1) and (2 * i + 2 >= n or is_min_heap(arr, n, 2 * i + 2))

'''
Utility functions for heap operations. Contains the function to check if an array is a min heap.
'''
def is_min_heap(arr, n, i):
    # If a leaf node
    if i >= (n - 2) // 2:
        return True
    # Check if the current node is smaller than its left child
    if arr[i] > arr[2 * i + 1]:
        return False
    # Check if the current node is smaller than its right child, if it exists
    if 2 * i + 2 < n and arr[i] > arr[2 * i + 2]:
        return False
    # Recursively check the left and right subtrees
    return is_min_heap(arr, n, 2 * i + 1) and (2 * i + 2 >= n or is_min_heap(arr, n, 2 * i + 2))

def is_min_heap(arr, n):
    # Iterate over all non-leaf nodes
    for i in range(n // 2):
        # Check if the current node is smaller than its left child
        if arr[i] > arr[2 * i + 1]:
            return False
        # Check if the current node is smaller than its right child, if it exists
        if 2 * i + 2 < n and arr[i] > arr[2 * i + 2]:
            return False
    return True

'''
Utility functions for heap operations. Contains the function to check if an array is a min heap.
'''
def is_min_heap(arr, n):
    # Iterate over all non-leaf nodes
    for i in range(n // 2):
        # Check if the current node is greater than its left child
        if arr[i] > arr[2 * i + 1]:
            return False
        # Check if the current node is greater than its right child, if it exists
        if 2 * i + 2 < n and arr[i] > arr[2 * i + 2]:
            return False
    return True

# Min Heap Checker

Min Heap Checker is a simple application designed to help users determine whether a given array represents a Min Heap. The application provides a user-friendly interface where users can input an array and receive immediate feedback on whether it satisfies the properties of a Min Heap.

## Quick Install

This application does not require any external dependencies beyond Python's standard library. To run the application, ensure you have Python installed on your system.

## How to Install

1. **Clone the Repository:**
   - Download or clone the repository to your local machine.

2. **Navigate to the Application Directory:**
   - Open a terminal or command prompt and navigate to the directory where the application files are located.

3. **Run the Application:**
   - Execute the following command to start the application:
     ```bash
     python main.py
     ```

## How to Use

1. **Launch the Application:**
   - After running `main.py`, a window titled "Min Heap Checker" will appear.

2. **Input the Array:**
   - Enter the elements of the array you wish to check in the input field. Elements should be separated by commas. For example: `1, 2, 3, 4, 5`.

3. **Check Min Heap:**
   - Click the "Check Min Heap" button to evaluate the array.

4. **View Results:**
   - The application will display a message indicating whether the array represents a Min Heap or not. If the input is invalid (e.g., non-integer values), an error message will be shown.

## Understanding Min Heap

A Min Heap is a complete binary tree where the value of each node is less than or equal to the values of its children. This application checks the array representation of a binary tree to ensure it satisfies this property.

## Additional Information

- **No External Dependencies:** The application is built using Python's standard library, specifically the `tkinter` module for the graphical user interface.
- **Source Code:** The logic for checking the Min Heap property is implemented in the `heap_utils.py` file.

For any further questions or support, please contact our development team. We hope you find the Min Heap Checker useful for your needs!
