'''

This file contains utility functions for the tuple converter application.
It includes the function to convert a tuple string to an integer tuple.

'''

def convert_tuple_string_to_int_tuple(tuple_string):
    '''
    Converts a tuple string to an integer tuple.
    
    Args:
        tuple_string (str): The input tuple string (e.g., "(1, 2, 3)").
    
    Returns:
        tuple: The converted integer tuple.
    
    Raises:
        ValueError: If the input string is not a valid tuple string.
    '''
    try:
        # Evaluate the string to a tuple
        result = eval(tuple_string)
        
        # Ensure the result is a tuple and contains only integers
        if isinstance(result, tuple) and all(isinstance(i, int) for i in result):
            return result
        else:
            raise ValueError("Input must be a tuple of integers.")
    except Exception as e:
        raise ValueError(f"Invalid tuple string: {e}")
```

'''

This file contains utility functions for the tuple converter application.

'''

def convert_tuple_string_to_int_tuple(tuple_string):
    '''
    Converts a tuple string to an integer tuple.
    
    Args:
        tuple_string (str): The input tuple string (e.g., "(1, 2, 3)").
    
    Returns:
        tuple: The converted integer tuple.
    
    Raises:
        ValueError: If the input string is not a valid tuple string.
    '''
    try:
        # Evaluate the string to a tuple
        result = eval(tuple_string)
        
        # Ensure the result is a tuple and contains only integers
        if isinstance(result, tuple) and all(isinstance(i, int) for i in result):
            return result
        else:
            raise ValueError("Input must be a tuple of integers.")
    except Exception as e:
        raise ValueError(f"Invalid tuple string: {e}")

def convert_tuple_string_to_int_tuple(tuple_string):
    '''
    Converts a tuple string to an integer tuple.
    Args:
        tuple_string (str): The input tuple string (e.g., "(1, 2, 3)").
    Returns:
        tuple: The converted integer tuple.
    Raises:
        ValueError: If the input string is not a valid tuple string.
    '''
    try:
        # Evaluate the string to a tuple
        result = eval(tuple_string)
        # Ensure the result is a tuple and contains only integers
        if isinstance(result, tuple) and all(isinstance(i, int) for i in result):
            return result
        else:
            raise ValueError("Input must be a tuple of integers.")
    except SyntaxError:
        raise ValueError("Invalid tuple string format.")
    except Exception as e:
        raise ValueError(f"Invalid tuple string: {e}")

def convert_tuple_string_to_int_tuple(tuple_string):
    '''
    Converts a tuple string to an integer tuple.
    Args:
        tuple_string (str): The input tuple string (e.g., "(1, 2, 3)").
    Returns:
        tuple: The converted integer tuple.
    Raises:
        ValueError: If the input string is not a valid tuple string.
    '''
    try:
        # Evaluate the string to a tuple
        result = eval(tuple_string)
        # Ensure the result is a tuple and contains only integers
        if isinstance(result, tuple) and all(isinstance(i, int) for i in result):
            return result
        else:
            raise ValueError("Input must be a tuple of integers.")
    except SyntaxError:
        raise ValueError("Invalid tuple string format.")

'''  
This file contains utility functions for the tuple converter application.  
It includes the function to convert a tuple string to an integer tuple.  
'''  
def convert_tuple_string_to_int_tuple(tuple_string):  
    '''  
    Converts a tuple string to an integer tuple.  
    Args:  
        tuple_string (str): The input tuple string (e.g., "(1, 2, 3)").  
    Returns:  
        tuple: The converted integer tuple.  
    Raises:  
        ValueError: If the input string is not a valid tuple string.  
    '''  
    try:  
        # Evaluate the string to a tuple  
        result = eval(tuple_string)  
        # Ensure the result is a tuple and contains only integers  
        if isinstance(result, tuple) and all(isinstance(i, int) for i in result):  
            return result  
        else:  
            raise ValueError("Input must be a tuple of integers.")  
    except SyntaxError:  
        raise ValueError("Invalid tuple string format.")  
    except Exception as e:  
        raise ValueError(f"Invalid tuple string: {e}")

## 🤔 What is this?

Large language models (LLMs) are emerging as a transformative technology, enabling developers to build applications that they previously could not. However, using these LLMs in isolation is often insufficient for creating a truly powerful app - the real power comes when you can combine them with other sources of computation or knowledge.

This library aims to assist in the development of those types of applications. Common examples of these applications include:

**❓ Question Answering over specific documents**

- Documentation

- End-to-end Example: Question Answering over Notion Database

# Tuple String to Integer Tuple Converter

Welcome to the Tuple String to Integer Tuple Converter application! This software provides a simple and intuitive graphical user interface (GUI) to convert tuple strings into integer tuples. It is designed to help users easily process tuple strings in Python format and convert them into usable integer tuples.

---

## Main Functions

### 1. **Tuple String Conversion**
The core functionality of this application is to convert a tuple string (e.g., `"(1, 2, 3)"`) into an integer tuple (e.g., `(1, 2, 3)`). The application ensures that the input is valid and raises appropriate error messages if the input format is incorrect.

### 2. **Graphical User Interface (GUI)**
The application features a user-friendly GUI built with `tkinter`. Users can input their tuple strings, click a button to convert them, and view the results directly in the interface.

### 3. **Error Handling**
The application includes robust error handling to ensure invalid inputs are flagged with clear error messages. This helps users understand and correct their input.

---

## Installation Guide

### Step 1: Install Python
Ensure you have Python installed on your system. This application requires Python version 3.6 or higher. You can download Python from [python.org](https://www.python.org/).

### Step 2: Clone the Repository
Download or clone the project repository to your local machine.

```bash
git clone <repository-url>
cd <repository-folder>
```

### Step 3: Install Dependencies
Install the required dependencies listed in the `requirements.txt` file. Run the following command in your terminal:

```bash
pip install -r requirements.txt
```

This will install all necessary libraries, including `tkinter` (which is included by default in most Python installations).

---

## How to Use the Application

### Step 1: Run the Application
Navigate to the project folder and execute the `main.py` file to start the application.

```bash
python main.py
```

### Step 2: Input Tuple String
Once the application is running, you will see a GUI window. Enter a tuple string in the input field. For example:

```
(1, 2, 3)
```

### Step 3: Convert Tuple
Click the "Convert" button to process the input. The application will convert the tuple string into an integer tuple and display the result below the input field.

### Step 4: View Results
The converted integer tuple will be displayed in the "Result" section of the GUI. If the input is invalid, an error message will be shown.

---

## Example Usage

### Valid Input
Input: `(1, 2, 3)`  
Output: `Result: (1, 2, 3)`

### Invalid Input
Input: `(1, "a", 3)`  
Output: Error message: "Invalid input: Input must be a tuple of integers."

---

## Troubleshooting

### Common Issues
1. **Invalid Tuple Format**
   - Ensure the input is in the correct tuple format, e.g., `(1, 2, 3)`.
   - Avoid using non-integer values or incorrect syntax.

2. **Dependency Issues**
   - Ensure all dependencies are installed using the `requirements.txt` file.
   - Verify your Python version is 3.6 or higher.

3. **Application Not Starting**
   - Check if `tkinter` is installed. It is included by default in most Python installations, but if missing, you may need to install it manually.
