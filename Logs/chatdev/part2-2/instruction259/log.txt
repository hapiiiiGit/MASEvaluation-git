'''

main.py

'''

import re
import tkinter as tk
from tkinter import messagebox

def starts_with_vowel(text):
    '''
    Check if the given string starts with a vowel using regex.
    
    Parameters:
    text (str): The string to check.
    
    Returns:
    bool: True if the string starts with a vowel, False otherwise.
    '''
    # Regex pattern to match a string starting with a vowel (case insensitive)
    pattern = r'^[aeiouAEIOU]'
    return bool(re.match(pattern, text))

class VowelCheckerApp:
    '''
    A simple GUI application to check if a string starts with a vowel.
    '''
    def __init__(self, master):
        '''
        Initialize the VowelCheckerApp with the given master window.
        
        Parameters:
        master (tk.Tk): The main window of the application.
        '''
        self.master = master
        master.title("Vowel Checker")

        self.label = tk.Label(master, text="Enter a string:")
        self.label.pack()

        self.entry = tk.Entry(master)
        self.entry.pack()

        self.check_button = tk.Button(master, text="Check", command=self.check_vowel)
        self.check_button.pack()

        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def check_vowel(self):
        '''
        Check if the input string starts with a vowel and update the result label.
        '''
        text = self.entry.get()
        if starts_with_vowel(text):
            result = "The string starts with a vowel."
        else:
            result = "The string does not start with a vowel."
        self.result_label.config(text=result)
        messagebox.showinfo("Result", result)

if __name__ == "__main__":
    root = tk.Tk()
    app = VowelCheckerApp(root)
    root.mainloop()
```

import re
import tkinter as tk
from tkinter import messagebox

def starts_with_vowel(text):
    '''
    Check if the given string starts with a vowel using regex.
    
    Parameters:
    text (str): The string to check.
    
    Returns:
    bool: True if the string starts with a vowel, False otherwise.
    '''
    # Regex pattern to match a string starting with a vowel (case insensitive)
    pattern = r'^[aeiouAEIOU]'
    return bool(re.match(pattern, text))

class VowelCheckerApp:
    '''
    A simple GUI application to check if a string starts with a vowel.
    '''
    def __init__(self, master):
        '''
        Initialize the VowelCheckerApp with the given master window.
        
        Parameters:
        master (tk.Tk): The main window of the application.
        '''
        self.master = master
        master.title("Vowel Checker")

        self.label = tk.Label(master, text="Enter a string:")
        self.label.pack()

        self.entry = tk.Entry(master)
        self.entry.pack()

        self.check_button = tk.Button(master, text="Check", command=self.check_vowel)
        self.check_button.pack()

        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def check_vowel(self):
        '''
        Check if the input string starts with a vowel and update the result label.
        '''
        text = self.entry.get()
        if starts_with_vowel(text):
            result = "The string starts with a vowel."
        else:
            result = "The string does not start with a vowel."
        self.result_label.config(text=result)
        messagebox.showinfo("Result", result)

if __name__ == "__main__":
    root = tk.Tk()
    app = VowelCheckerApp(root)
    root.mainloop()

import re
import tkinter as tk
from tkinter import messagebox
def starts_with_vowel(text):
    '''
    Check if the given string starts with a vowel using regex.
    Parameters:
    text (str): The string to check.
    Returns:
    bool: True if the string starts with a vowel, False otherwise.
    '''
    # Regex pattern to match a string starting with a vowel (case insensitive)
    pattern = r'^[aeiouAEIOU]'
    return bool(re.match(pattern, text))
class VowelCheckerApp:
    '''
    A simple GUI application to check if a string starts with a vowel.
    '''

        '''
        Initialize the VowelCheckerApp with the given master window.
        Parameters:
        master (tk.Tk): The main window of the application.
        '''

        master.title("Vowel Checker")
        self.label = tk.Label(master, text="Enter a string:")
        self.label.pack()
        self.entry = tk.Entry(master)
        self.entry.pack()
        self.check_button = tk.Button(master, text="Check", command=self.check_vowel)
        self.check_button.pack()
        self.result_label = tk.Label(master, text="")
        self.result_label.pack()
    def check_vowel(self):
        '''
        Check if the input string starts with a vowel and update the result label.

        if starts_with_vowel(text):
            result = "The string starts with a vowel."
        else:
            result = "The string does not start with a vowel."
        self.result_label.config(text=result)
        messagebox.showinfo("Result", result)
if __name__ == "__main__":
    root = tk.Tk()

Welcome to the Vowel Checker Application! This software is designed to help you determine whether a given string starts with a vowel using regular expressions. The application is built using Python and provides a simple graphical user interface (GUI) for ease of use.

## Main Functions

### Vowel Checking Function

The core functionality of this application is to check if a string starts with a vowel. This is achieved using a regular expression pattern that matches strings starting with any vowel (a, e, i, o, u) in both uppercase and lowercase.

### Graphical User Interface (GUI)

The application includes a user-friendly GUI built with Tkinter, a standard Python library for creating desktop applications. The GUI allows users to input a string and check if it starts with a vowel, displaying the result in a clear and concise manner.

## Installation

### Environment Setup

To run the Vowel Checker Application, you need to have Python installed on your system. The application does not require any external dependencies beyond the standard Python library.

### Steps to Install

1. **Install Python**: If you haven't already installed Python, download and install it from the [official Python website](https://www.python.org/downloads/).

2. **Download the Application**: Obtain the `main.py` file from the project repository or source.

3. **Run the Application**: Open a terminal or command prompt, navigate to the directory containing `main.py`, and execute the following command:

   ```bash
   python main.py
   ```

This will launch the Vowel Checker Application with its GUI.

## How to Use

1. **Launch the Application**: Run the application using the command mentioned above.

2. **Enter a String**: In the application window, you will see a text entry field labeled "Enter a string:". Type the string you want to check into this field.

3. **Check the String**: Click the "Check" button. The application will evaluate whether the entered string starts with a vowel.

4. **View Results**: The result will be displayed in the application window, indicating whether the string starts with a vowel or not. Additionally, a message box will pop up with the result for your convenience.

## Conclusion

The Vowel Checker Application is a simple yet effective tool for quickly determining if a string starts with a vowel. With its intuitive GUI and straightforward functionality, it is accessible to users of all levels of technical expertise. Enjoy using the application to enhance your text processing tasks!
