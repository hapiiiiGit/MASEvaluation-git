'''

This module contains the main application code for a GUI that calculates the sum and average of the first n natural numbers.

'''

import tkinter as tk
from tkinter import messagebox

class SumAndAverageCalculator:
    '''
    A class to calculate the sum and average of the first n natural numbers.
    '''
    @staticmethod
    def calculate_sum_and_average(n):
        '''
        Calculate the sum and average of the first n natural numbers.

        :param n: The number of natural numbers to consider.
        :return: A tuple containing the sum and average.
        '''
        if n < 1:
            raise ValueError("n must be a positive integer")
        total_sum = n * (n + 1) // 2
        average = total_sum / n
        return total_sum, average

class MainApplication:
    '''
    A class to create the main application window and handle user interaction.
    '''
    def __init__(self, master):
        '''
        Initialize the main application window.

        :param master: The root window.
        '''
        self.master = master
        master.title("Sum and Average Calculator")

        self.label = tk.Label(master, text="Enter a positive integer:")
        self.label.pack()

        self.entry = tk.Entry(master)
        self.entry.pack()

        self.calculate_button = tk.Button(master, text="Calculate", command=self.calculate)
        self.calculate_button.pack()

        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def calculate(self):
        '''
        Handle the calculation and display the result.
        '''
        try:
            n = int(self.entry.get())
            calculator = SumAndAverageCalculator()
            total_sum, average = calculator.calculate_sum_and_average(n)
            self.result_label.config(text=f"Sum: {total_sum}, Average: {average:.2f}")
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))

if __name__ == "__main__":
    root = tk.Tk()
    app = MainApplication(root)
    root.mainloop()
```

This code sets up a simple GUI application using `tkinter`. The user can enter a positive integer, and upon clicking the "Calculate" button, the application will display the sum and average of the first n natural numbers. The `SumAndAverageCalculator` class handles the mathematical logic, while the `MainApplication` class manages the GUI.

'''

This module contains the main application code for a GUI that calculates the sum and average of the first n natural numbers.

'''

import tkinter as tk
from tkinter import messagebox

class SumAndAverageCalculator:
    '''
    A class to calculate the sum and average of the first n natural numbers.
    '''
    @staticmethod
    def calculate_sum_and_average(n):
        '''
        Calculate the sum and average of the first n natural numbers.

        :param n: The number of natural numbers to consider.
        :return: A tuple containing the sum and average.
        '''
        if n < 1:
            raise ValueError("n must be a positive integer")
        total_sum = n * (n + 1) // 2
        average = total_sum / n
        return total_sum, average

class MainApplication:
    '''
    A class to create the main application window and handle user interaction.
    '''
    def __init__(self, master):
        '''
        Initialize the main application window.

        :param master: The root window.
        '''
        self.master = master
        master.title("Sum and Average Calculator")

        self.label = tk.Label(master, text="Enter a positive integer:")
        self.label.pack()

        self.entry = tk.Entry(master)
        self.entry.pack()

        self.calculate_button = tk.Button(master, text="Calculate", command=self.calculate)
        self.calculate_button.pack()

        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def calculate(self):
        '''
        Handle the calculation and display the result.
        '''
        try:
            n = int(self.entry.get())
            calculator = SumAndAverageCalculator()
            total_sum, average = calculator.calculate_sum_and_average(n)
            self.result_label.config(text=f"Sum: {total_sum}, Average: {average:.2f}")
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))

if __name__ == "__main__":
    root = tk.Tk()
    app = MainApplication(root)
    root.mainloop()
```

This module contains the main application code for a GUI that calculates the sum and average of the first n natural numbers.
'''
import tkinter as tk
from tkinter import messagebox

class SumAndAverageCalculator:
    '''
    A class to calculate the sum and average of the first n natural numbers.
    '''
    @staticmethod
    def calculate_sum_and_average(n):
        '''
        Calculate the sum and average of the first n natural numbers.
        :param n: The number of natural numbers to consider.
        :return: A tuple containing the sum and average.
        '''
        if n < 1:
            raise ValueError("n must be a positive integer")
        total_sum = n * (n + 1) // 2
        average = total_sum / n
        return total_sum, average

class MainApplication:
    '''
    A class to create the main application window and handle user interaction.
    '''
    def __init__(self, master):
        '''
        Initialize the main application window.
        :param master: The root window.
        '''
        self.master = master
        master.title("Sum and Average Calculator")
        self.label = tk.Label(master, text="Enter a positive integer:")
        self.label.pack()
        self.entry = tk.Entry(master)
        self.entry.pack()
        self.calculate_button = tk.Button(master, text="Calculate", command=self.calculate)
        self.calculate_button.pack()
        self.result_label = tk.Label(master, text="")
        self.result_label.pack()

    def calculate(self):
        '''
        Handle the calculation and display the result.
        '''
        try:
            n = int(self.entry.get())
            calculator = SumAndAverageCalculator()
            total_sum, average = calculator.calculate_sum_and_average(n)
            self.result_label.config(text=f"Sum: {total_sum}, Average: {average:.2f}")
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))

if __name__ == "__main__":
    root = tk.Tk()
    app = MainApplication(root)
    root.mainloop()

This module contains the main application code for a GUI that calculates the sum and average of the first n natural numbers.

import tkinter as tk
from tkinter import messagebox
class SumAndAverageCalculator:
    '''
    A class to calculate the sum and average of the first n natural numbers.
    '''
    @staticmethod
    def calculate_sum_and_average(n):
        '''
        Calculate the sum and average of the first n natural numbers.
        :param n: The number of natural numbers to consider.
        :return: A tuple containing the sum and average.
        '''
        if n < 1:
            raise ValueError("n must be a positive integer")
        total_sum = n * (n + 1) // 2
        average = total_sum / n
        return total_sum, average
class MainApplication:
    '''
    A class to create the main application window and handle user interaction.
    '''
    def __init__(self, master):
        '''
        Initialize the main application window.
        :param master: The root window.
        '''
        self.master = master
        master.title("Sum and Average Calculator")
        self.label = tk.Label(master, text="Enter a positive integer:")
        self.label.pack()
        self.entry = tk.Entry(master)
        self.entry.pack()
        self.calculate_button = tk.Button(master, text="Calculate", command=self.calculate)
        self.calculate_button.pack()
        self.result_label = tk.Label(master, text="")
        self.result_label.pack()
    def calculate(self):
        '''
        Handle the calculation and display the result.
        '''
        try:
            n = int(self.entry.get())
            calculator = SumAndAverageCalculator()
            total_sum, average = calculator.calculate_sum_and_average(n)
            self.result_label.config(text=f"Sum: {total_sum}, Average: {average:.2f}")
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))
if __name__ == "__main__":
    root = tk.Tk()
    app = MainApplication(root)
    root.mainloop()

import tkinter as tk
from tkinter import messagebox
class SumAndAverageCalculator:
    '''
    A class to calculate the sum and average of the first n natural numbers.
    '''
    @staticmethod
    def calculate_sum_and_average(n):
        '''
        Calculate the sum and average of the first n natural numbers.
        :param n: The number of natural numbers to consider.
        :return: A tuple containing the sum and average.
        '''
        if n < 1:
            raise ValueError("n must be a positive integer")
        total_sum = n * (n + 1) // 2
        average = total_sum / n
        return total_sum, average
class MainApplication:
    '''
    A class to create the main application window and handle user interaction.
    '''
    def __init__(self, master):
        '''
        Initialize the main application window.
        :param master: The root window.
        '''
        self.master = master
        master.title("Sum and Average Calculator")
        self.label = tk.Label(master, text="Enter a positive integer:")
        self.label.pack()
        self.entry = tk.Entry(master)
        self.entry.pack()
        self.calculate_button = tk.Button(master, text="Calculate", command=self.calculate)
        self.calculate_button.pack()
        self.result_label = tk.Label(master, text="")
        self.result_label.pack()
    def calculate(self):
        '''
        Handle the calculation and display the result.
        '''
        try:
            n = int(self.entry.get())
            calculator = SumAndAverageCalculator()
            total_sum, average = calculator.calculate_sum_and_average(n)
            self.result_label.config(text=f"Sum: {total_sum}, Average: {average:.2f}")
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))

Welcome to the Sum and Average Calculator! This software is designed to help you calculate the sum and average of the first n natural numbers using a simple graphical user interface (GUI). Below, you will find detailed instructions on how to install, set up, and use the application.

## Introduction

The Sum and Average Calculator is a Python-based application that allows users to input a positive integer and receive the sum and average of the first n natural numbers. This application is ideal for educational purposes, quick calculations, and understanding basic arithmetic operations.

## Installation

### Environment Setup

To run the Sum and Average Calculator, you need to have Python installed on your system. The application does not require any external dependencies, making it easy to set up and use.

1. **Install Python**: If you haven't already installed Python, download and install it from the official [Python website](https://www.python.org/downloads/).

2. **Verify Installation**: Open your terminal or command prompt and type `python --version` to ensure Python is installed correctly.

### Application Setup

1. **Download the Code**: Obtain the `main.py` file from the provided source.

2. **Run the Application**: Navigate to the directory containing `main.py` using your terminal or command prompt. Execute the following command to start the application:

   ```bash
   python main.py
   ```

## Usage

### Main Functions

The application provides a simple GUI for calculating the sum and average of the first n natural numbers. Here are the main functions:

- **Input Field**: Enter a positive integer in the input field provided in the application window.

- **Calculate Button**: Click the "Calculate" button to compute the sum and average.

- **Result Display**: The sum and average will be displayed in the application window.

### Step-by-Step Guide

1. **Launch the Application**: Run `main.py` as described in the installation section.

2. **Enter a Positive Integer**: In the application window, you will see a label prompting you to enter a positive integer. Type the desired number into the entry field.

3. **Calculate**: Click the "Calculate" button. The application will process your input and display the sum and average of the first n natural numbers.

4. **View Results**: The results will appear below the "Calculate" button, showing the sum and average.

5. **Error Handling**: If you enter an invalid input (e.g., a non-integer or negative number), an error message will be displayed, prompting you to enter a valid positive integer.

## Troubleshooting

- **Invalid Input Error**: Ensure you enter a positive integer. The application will not accept negative numbers or non-integer values.

- **Application Not Starting**: Verify that Python is installed correctly and that you are running the command in the directory containing `main.py`.

## Conclusion

The Sum and Average Calculator is a straightforward tool for performing basic arithmetic operations on natural numbers. With its user-friendly interface, it is accessible to users of all levels. Enjoy using the application for your calculations!
