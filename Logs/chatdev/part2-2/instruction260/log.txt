'''

This module contains the logic for calculating the geometric sum of n-1 using recursion.

'''

def calculate_geometric_sum(n):
    """
    Calculate the geometric sum of n-1 using recursion.
    
    :param n: The number of terms in the geometric series.
    :return: The geometric sum of n-1.
    """
    if n < 1:
        return 0
    else:
        return 1 / (2 ** (n - 1)) + calculate_geometric_sum(n - 1)

def calculate_geometric_sum(n):
    """
    Calculate the geometric sum of n-1 using recursion.

def calculate_geometric_sum(n):
    """
    Calculate the geometric sum of n-1 using recursion.
    :param n: The number of terms in the geometric series.
    :return: The geometric sum of n-1.
    """
    if n < 1:

# Geometric Sum Calculator User Manual

Welcome to the Geometric Sum Calculator application! This software is designed to calculate the geometric sum of n-1 using recursion. The application provides a simple graphical user interface (GUI) for users to input their desired value of n and receive the calculated geometric sum.

## Main Functions of the Software

### Geometric Sum Calculation

The core functionality of this application is to compute the geometric sum of n-1 using a recursive approach. The geometric sum is calculated as follows:

\[ \text{Geometric Sum} = \frac{1}{2^{n-1}} + \frac{1}{2^{n-2}} + \ldots + \frac{1}{2^0} \]

### Graphical User Interface

The application features a user-friendly GUI built with Tkinter, allowing users to easily input their desired value of n and view the calculated geometric sum.
