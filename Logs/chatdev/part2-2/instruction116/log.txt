#### 2. `shell_sort.py`

```python
'''

Shell Sort algorithm implementation.

'''

def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    
    return arr
```

#### 3. `utils.py`

```python
'''

Utility functions for input parsing and output formatting.

def parse_input(input_str):
    try:
        return [int(x.strip()) for x in input_str.split(',')]
    except ValueError:
        raise ValueError("Please enter a valid list of integers separated by commas.")

def format_output(arr):
    return ', '.join(map(str, arr))

'''

Shell Sort algorithm implementation.

'''

def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    
    return arr
```

'''
Shell Sort algorithm implementation.
'''
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

'''
Shell Sort algorithm implementation.
'''
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

'''
Shell Sort algorithm implementation.
'''
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

'''
Main application file for Shell Sort GUI using tkinter.
'''
import tkinter as tk
from tkinter import messagebox
from shell_sort import shell_sort
from utils import parse_input, format_output
class ShellSortApp:
    def __init__(self, root):
        self.root = root
        self.root.title("Shell Sort Application")
        self.label = tk.Label(root, text="Enter numbers separated by commas:")
        self.label.pack()
        self.entry = tk.Entry(root, width=50)
        self.entry.pack()
        self.sort_button = tk.Button(root, text="Sort", command=self.sort_array)
        self.sort_button.pack()
        self.result_label = tk.Label(root, text="Sorted Array:")
        self.result_label.pack()
        self.result_display = tk.Label(root, text="", fg="blue")
        self.result_display.pack()
    def sort_array(self):
        input_data = self.entry.get()
        try:
            array = parse_input(input_data)
            sorted_array = shell_sort(array)
            self.result_display.config(text=format_output(sorted_array))
        except ValueError as e:
            messagebox.showerror("Invalid input", str(e))
if __name__ == "__main__":
    root = tk.Tk()
    app = ShellSortApp(root)
    root.mainloop()

'''
Shell Sort algorithm implementation.
'''
def shell_sort(arr):
    n = len(arr)
    gap = n // 2
    while gap > 0:
        for i in range(gap, n):
            temp = arr[i]
            j = i
            while j >= gap and arr[j - gap] > temp:
                arr[j] = arr[j - gap]
                j -= gap
            arr[j] = temp
        gap //= 2
    return arr

'''
Utility functions for input parsing and output formatting.
'''
def parse_input(input_str):
    try:
        return [int(x.strip()) for x in input_str.split(',')]
    except ValueError:
        raise ValueError("Please enter a valid list of integers separated by commas.")
def format_output(arr):
    return ', '.join(map(str, arr))

Welcome to the Shell Sort Application! This application provides a simple graphical user interface (GUI) to sort arrays using the Shell Sort algorithm. It is designed to be user-friendly and requires no external dependencies.

## 🤔 What is this?

The Shell Sort Application is a tool that allows users to input a list of numbers and sort them using the Shell Sort algorithm. Shell Sort is an efficient sorting algorithm that is a generalization of insertion sort, allowing the exchange of items that are far apart. This application provides a visual and interactive way to understand and utilize this sorting technique.

## Main Functions

- **Input Parsing**: Users can enter a list of numbers separated by commas. The application will parse this input and convert it into a list of integers.
- **Sorting**: The core functionality of the application is to sort the input list using the Shell Sort algorithm.
- **Output Formatting**: The sorted list is displayed in a user-friendly format.

## How to Use the Application

1. **Launch the Application**: Run the `main.py` file to start the application. This will open a window with the Shell Sort GUI.

2. **Enter Numbers**: In the provided text entry field, input the numbers you wish to sort, separated by commas (e.g., `34, 7, 23, 32, 5, 62`).

3. **Sort the Array**: Click the "Sort" button. The application will process the input, sort the numbers using the Shell Sort algorithm, and display the sorted array.

4. **View Results**: The sorted array will be displayed below the "Sorted Array:" label in the application window.

## Error Handling

- If the input is invalid (e.g., contains non-numeric characters or is not properly formatted), the application will display an error message prompting the user to enter a valid list of integers.

## Example

1. **Input**: `45, 12, 78, 3, 56`
2. **Output**: `3, 12, 45, 56, 78`

## Documentation
