Main application file for the whitespace and underscore replacement tool using tkinter.

class MainApp:
    def __init__(self, root):
        '''
        Initialize the main application with GUI components.

import tkinter as tk
from utils import replace_whitespace_underscore
class MainApp:
    def __init__(self, root):
        '''
        Initialize the main application with GUI components.
        Parameters:
        root (tk.Tk): The root window of the tkinter application.
        '''
        self.root = root
        self.root.title("Whitespace and Underscore Replacer")
        self.root.geometry("400x200")  # Set a fixed window size
        # Label for input prompt
        self.label = tk.Label(root, text="Enter your string:")
        self.label.pack(pady=10)
        # Entry widget for user input
        self.entry = tk.Entry(root, width=50)
        self.entry.pack(pady=5)
        # Button to trigger replacement
        self.button = tk.Button(root, text="Replace", command=self.replace_characters)
        self.button.pack(pady=10)
        # Label to display the result
        self.result_label = tk.Label(root, text="", wraplength=350, justify="left")
        self.result_label.pack(pady=10)
    def replace_characters(self):
        '''

        input_string = self.entry.get()

# Whitespace and Underscore Replacer

A simple application to replace whitespaces with underscores and vice versa in a given string using a graphical user interface (GUI) built with Python's `tkinter` library.

---

## 📖 Introduction

The **Whitespace and Underscore Replacer** is a lightweight tool designed to simplify the process of swapping whitespaces and underscores in text strings. Whether you're working with filenames, text formatting, or any other string manipulation task, this application provides an intuitive interface to perform the transformation.

---

## 🚀 Main Features

1. **String Transformation**: Replace whitespaces (`' '`) with underscores (`'_'`) and vice versa in any input string.
2. **User-Friendly GUI**: Built with `tkinter`, the application offers a clean and simple interface for users to input strings and view results.
3. **Error Handling**: Alerts users if the input string is empty or invalid.
4. **Lightweight**: No third-party dependencies are required; the application uses Python's standard library.

---

## 🛠️ Installation Guide

### Prerequisites

- Python version **3.6 or higher** installed on your system.

### Steps to Install

1. Clone or download the project repository to your local machine.
2. Ensure Python is installed and accessible via your terminal or command prompt.
3. Navigate to the project directory.
4. No additional dependencies are required, but you can verify the Python version by running:
   ```bash
   python --version
   ```
5. The application is ready to run without further setup.

---

## 🎮 How to Use the Application

### Running the Application

1. Open your terminal or command prompt.
2. Navigate to the directory containing the project files.
3. Run the application using the following command:
   ```bash
   python main.py
   ```

### Using the Application

1. **Input a String**: Enter the string you want to modify in the text box provided.
2. **Replace Characters**: Click the "Replace" button to perform the transformation.
3. **View Results**: The modified string will be displayed below the button.
4. **Error Handling**: If the input string is empty, an error message will be shown.

---

## 🧩 Example Usage

```
Hello_World
```

### Output:
```
Hello World
```

```

---

## 📂 Project Structure

- **main.py**: The main application file containing the GUI logic.
- **utils.py**: A utility file with the core function for string manipulation.
- **requirements.txt**: Specifies the Python version required for the project.

---

## 🔧 Technical Details

### Core Functionality

The application uses the `replace_whitespace_underscore` function from `utils.py` to perform the transformation. This function swaps whitespaces and underscores using a list comprehension for efficient character-by-character processing.

### GUI Design

The GUI is built using `tkinter`, a standard Python library for creating graphical interfaces. It includes:
- Input text box for user strings.
- A button to trigger the replacement process.
- A label to display the result or error messages.

---

## 🛡️ Troubleshooting

### Common Issues

1. **Empty Input**: Ensure you enter a valid string in the text box before clicking "Replace."
2. **Python Compatibility**: Verify that your Python version is 3.6 or higher.
