def sum_of_products_of_sublists(lst):
    total_sum = 0
    n = len(lst)
    for i in range(n):
        product = 1
        for j in range(i, n):
            product *= lst[j]
            total_sum += product
    return total_sum

# Sum of Products of Sublists Calculator

## Quick Install

To get started with the Sum of Products of Sublists Calculator, you need to have Python installed on your system. The application is compatible with Python version 3.6 and above.

### Installation Steps

1. **Clone the Repository:**

   First, clone the repository to your local machine using the following command:

   ```bash
   git clone <repository-url>
   ```

   Replace `<repository-url>` with the actual URL of the repository.

2. **Navigate to the Project Directory:**

   Change your directory to the project folder:

   ```bash
   cd <project-directory>
   ```

   Replace `<project-directory>` with the name of the cloned repository folder.

3. **Install Python:**

   Ensure you have Python installed. You can download it from [Python's official website](https://www.python.org/downloads/).

4. **Install Dependencies:**

   This project does not require any external dependencies. However, ensure your Python version is 3.6 or above. You can check your Python version with:

   ```bash
   python --version
   ```

## 🤔 What is this?

The Sum of Products of Sublists Calculator is a simple application that computes the sum of products of all possible sublists of a given list of integers. This is particularly useful in mathematical computations and analysis where such operations are needed.

## 📖 How to Use the Application

   Execute the following command in your terminal to start the application:

   ```bash
   python main.py
   ```

   Once the application window opens, you will see a text entry field. Enter a list of numbers separated by commas. For example:

   ```
   1,2,3
