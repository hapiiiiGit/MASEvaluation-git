def count_distinct_characters(string: str) -> int:
    """ Given a string, find out how many distinct characters (regardless of case) does it consist of
    >>> count_distinct_characters('xyzXYZ')
    3
    >>> count_distinct_characters('Jerry')
    4
    """

'''
def count_distinct_characters(string: str) -> int:
    """ Given a string, find out how many distinct characters (regardless of case) does it consist of
    >>> count_distinct_characters('xyzXYZ')
    3
    >>> count_distinct_characters('Jerry')
    4
    """
    # Convert the string to lowercase to ignore case
    string = string.lower()
    # Use a set to store unique characters
    unique_chars = set(string)
    # Return the count of unique characters
    return len(unique_chars)

import tkinter as tk

# Distinct Characters Counter

## Introduction

Welcome to the Distinct Characters Counter application! This application allows you to input a string and find out how many distinct characters (regardless of case) it consists of.

## Main Functions

- Input a string
- Click on the "Count Distinct Characters" button
- See the number of distinct characters in the provided string

## Installation

To use the Distinct Characters Counter application, you need to have Python installed on your system. You can install the required dependencies by running the following command in your terminal:

```bash
pip install -r requirements.txt
```

## How to Use

1. Clone or download the repository to your local machine.
2. Navigate to the directory where you saved the files.
3. Run the `gui.py` file using the following command:

```bash
python gui.py
```

4. A GUI window will open with a text entry field.
5. Enter a string in the text entry field.
6. Click on the "Count Distinct Characters" button.
7. The application will display the number of distinct characters in the provided string.

## Example

For example, if you enter the string 'xyzXYZ', the application will output '3' as there are 3 distinct characters in the string.
