import tkinter as tk
from tkinter import messagebox
from logic import will_it_fly

class App:
    def __init__(self, root):
        self.root = root
        self.root.title("Will It Fly?")
        
        self.label_q = tk.Label(root, text="Enter list of weights (comma-separated):")
        self.label_q.pack()
        
        self.entry_q = tk.Entry(root)
        self.entry_q.pack()
        
        self.label_w = tk.Label(root, text="Enter maximum weight:")
        self.label_w.pack()
        
        self.entry_w = tk.Entry(root)
        self.entry_w.pack()
        
        self.button = tk.Button(root, text="Check", command=self.check_fly)
        self.button.pack()
        
        self.result_label = tk.Label(root, text="")
        self.result_label.pack()

    def check_fly(self):
        try:
            q = list(map(int, self.entry_q.get().split(',')))
            w = int(self.entry_w.get())
            result = will_it_fly(q, w)
            self.result_label.config(text="Will it fly? " + str(result))
        except ValueError:
            messagebox.showerror("Input Error", "Please enter valid integers.")

if __name__ == "__main__":
    root = tk.Tk()
    app = App(root)
    root.mainloop()

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    
    # Check if the sum of the elements is less than or equal to w
    if sum(q) <= w:
        return True
    else:
        return False

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) <= w:
        return True
    else:
        return False

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) <= w:
        return True
    else:
        return False

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) <= w:
        return True
    else:
        return False

    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal to the maximum possible weight w.
    '''

        return False

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal to the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) > w:
        return False
    return True

class App:
    def __init__(self, root):

        self.label_q = tk.Label(root, text="Enter list of weights (comma-separated):")

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal to the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) > w:
        return False
    return True

def will_it_fly(q, w):
    '''
    Write a function that returns True if the object q will fly, and False otherwise.
    The object q will fly if it's balanced (it is a palindromic list) and the sum of its elements
    is less than or equal to the maximum possible weight w.
    '''
    # Check if the list is a palindrome
    if q != q[::-1]:
        return False
    # Check if the sum of the elements is less than or equal to w
    if sum(q) > w:
        return False
    return True

# Will It Fly?

A simple application to determine if an object will fly based on its weight distribution and maximum allowable weight.

## Introduction

The "Will It Fly?" application allows users to input a list of weights and a maximum weight limit to check if the object is balanced (palindromic list) and if its total weight is within the specified limit. The application provides a user-friendly interface using Tkinter, making it easy to interact with.

## Main Functions

- **Input Weights**: Users can enter a list of weights separated by commas.
- **Input Maximum Weight**: Users can specify the maximum allowable weight.
- **Check Functionality**: The application checks if the object will fly based on the criteria:
  - The list of weights must be a palindrome.
  - The sum of the weights must be less than or equal to the maximum weight.

## Installation

To run the "Will It Fly?" application, you need to have Python installed on your machine. Follow these steps to set up the environment:

1. **Install Python**: Download and install Python from the [official website](https://www.python.org/downloads/).

2. **Install Tkinter**: Tkinter is included with most Python installations. If you don't have it, you can install it using:
   ```bash
   sudo apt-get install python3-tk  # For Ubuntu
   ```

3. **Clone the Repository**: Clone the repository containing the application files.
   ```bash
   git clone <repository-url>
   cd <repository-directory>
   ```

4. **Install Dependencies**: If there are any additional dependencies, you can install them using pip. For this application, no external libraries are required beyond the standard library.

## How to Use the Application

1. **Run the Application**: Navigate to the directory containing the `main.py` file and run the following command:
   ```bash
   python main.py
   ```

2. **Input Data**:
   - In the application window, enter a list of weights in the first input field (e.g., `3,2,3`).
   - Enter the maximum weight in the second input field (e.g., `9`).

3. **Check if it Will Fly**: Click the "Check" button to see if the object will fly based on the provided inputs.

4. **View Results**: The application will display whether the object will fly or not based on the criteria.

## Example Usage

- **Example 1**: Input: `1,2`, Maximum Weight: `5`
  - Output: `Will it fly? False` (Unbalanced)
  
- **Example 2**: Input: `3,2,3`, Maximum Weight: `1`
  - Output: `Will it fly? False` (Exceeds maximum weight)
  
- **Example 3**: Input: `3,2,3`, Maximum Weight: `9`
  - Output: `Will it fly? True` (Balanced and within weight limit)
  
- **Example 4**: Input: `3`, Maximum Weight: `5`
  - Output: `Will it fly? True` (Single weight is balanced and within limit)

## Conclusion

The "Will It Fly?" application provides a straightforward way to determine if an object is balanced and within weight limits. With its simple interface and clear functionality, users can easily assess their inputs and get immediate feedback.
