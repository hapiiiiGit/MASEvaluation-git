# Critique:
# The given solution is correct in terms of logic and will produce the correct sum of XOR of all pairs.
# However, it has O(n^2) time complexity which might be inefficient for large lists.
# We can optimize the solution using bitwise properties:
# For each bit position, count how many numbers have that bit set and how many don't.
# The contribution of that bit to the XOR sum of all pairs is count_set * count_unset * (1 << bit_position).
# Summing this over all bit positions gives the total sum of XOR of all pairs in O(n * log(max_num)) time.

def sum_xor_pairs(nums):
    max_num = max(nums) if nums else 0
    max_bit = max_num.bit_length()
    result = 0
    n = len(nums)
    for bit in range(max_bit):
        count_set = sum((num >> bit) & 1 for num in nums)
        count_unset = n - count_set
        result += count_set * count_unset * (1 << bit)
    return result
